
# Building Blochs
This Physical representation of a [bloch sphere](https://en.wikipedia.org/wiki/Bloch_sphere) was developed by Quantum Delta NL and The Technology Center at the University of Amsterdam for various outreach and educational purposes. The files and documentation on this repository should be sufficient for anyone to build this model and get it running.
